# 4649.net - よろしくねっ 🌈

日本語の語呂合わせ「よろしく（4649）」をドメインにしたウェブサイトです。

## 🌐 公開URL

- **本番サイト**: [https://4649.net](https://4649.net)（DNS反映待ち）
- **Vercel URL**: [https://4649-net.vercel.app](https://4649-net.vercel.app)

## ✨ 特徴

- サイケデリックなグラデーション背景
- 波紋エフェクトアニメーション
- マウスフォロワーエフェクト
- クリックでサブタイトル変更
- レスポンシブ対応

## 🛠 技術スタック

- HTML5
- CSS3（カスタムアニメーション）
- Vanilla JavaScript
- Google Fonts（Noto Sans JP）

## 📦 デプロイ

このプロジェクトは **Vercel** でホスティングされています。  
`main` ブランチにプッシュすると自動デプロイされます。
